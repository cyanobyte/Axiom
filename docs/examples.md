# Examples

The repo includes a small example set with different goals.

## Beginner Web Example

File:

- `examples/basic/counter-webapp.axiom.js`

Use it for:

- first deterministic build
- seeing the basic authored intent shape
- understanding isolated generated output

Related config:

- `examples/basic/axiom.config.js`

## CLI Example

File:

- `examples/cli/echo-tool.axiom.js`

Use it for:

- compact-mode authoring
- a small non-web workflow
- `analyze` and `fix` experiments

Related config:

- `examples/cli/axiom.config.js`

## Live Smoke Example

File:

- `examples/live-counter/counter-webapp.axiom.js`

Use it for:

- manual live-provider verification
- end-to-end generated app behavior
- acceptance-level smoke testing

Related config:

- `examples/live-counter/axiom.config.js`

## Dogfood Slice Example

File:

- `examples/dogfood/axiom-runtime-slice.axiom.js`

Use it for:

- safe early dogfooding
- an Axiom-adjacent generated slice without mutating the real source tree

Related config:

- `examples/dogfood/axiom.config.js`

## Internal Richer Examples

The `docs/superpowers/examples/` directory contains richer internal examples used in design and planning work.
